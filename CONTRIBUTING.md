# Contributing to RefreshFlow

First off, thank you for taking the time to contribute! We welcome contributions of all forms: bug fixes, feature enhancements, documentation updates, and design suggestions.

## Development Setup

1. **Prerequisites**: Make sure you have Node.js (v18+) and npm installed.
2. **Clone and Install**:
   ```bash
   git clone https://github.com/your-username/refreshflow.git
   cd refreshflow
   npm install
   ```
3. **Build the extension**:
   ```bash
   npm run build
   ```
4. **Load in Chrome**:
   - Open Chrome and navigate to `chrome://extensions/`.
   - Enable "Developer mode" in the top right.
   - Click "Load unpacked" and select the `dist` directory generated by the build process.

## Code Standards

- **TypeScript**: We enforce strict type checking. Ensure your code compiles without type errors.
- **Formatting**: We use Prettier for code formatting. Run `npm run format` before committing.
- **Linting**: Run `npm run lint` to ensure there are no lint errors.
- **Testing**: Write unit tests for utility functions and logic. Run tests with `npm test`.

## Pull Request Process

1. Create a branch for your feature or bug fix: `git checkout -b feature/my-new-feature` or `git checkout -b bugfix/issue-description`.
2. Commit your changes with clear, descriptive commit messages.
3. Push to your branch and open a Pull Request.
4. Ensure all CI checks (lint, typecheck, test, build) pass.
5. A project maintainer will review your code.
